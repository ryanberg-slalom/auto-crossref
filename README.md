# Auto Crossref

A web app for exploring autocross event results and driver statistics.

## Getting Started

```bash
cd app
npm install
npm run dev
```

## Adding a New Event

Place the three Pronto Timing System PDFs (PAX, RAW, FINAL) into a new folder under `data/{year}/`, then ask GitHub Copilot to add the event data. It knows the full process.

> **Example prompt:** "Add latest events"
